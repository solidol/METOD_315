\chapter{Масиви у PHP}

\section*{Анотація} 
У той час як скалярна змінна PHP зберігає одне значення, змінну масиву можна використовувати для зберігання безлічі або послідовності значень. система PHP підтримує масиви з числовими індексами і асоціативні масиви. 

\section*{План лекції}
\begin{enumerate}
\item Масиви з числовими індексами
\item Асоціативні масиви
\item Функції для роботи з масивами


\end{enumerate}

\section{Масиви з числовими індексами}
 \$my\_array = array ( 'red', 'green', 'blue'); 

Цей код створює масив з числовим індексом з ім'ям \$my\_array. Масиву присвоюється три елементи - red, green, і blue. Кожен елемент ідентифікується числовим індексом.

 \$my\_array [0] = 'red' // індекс 0 відповідає елементу red
 
 \$my\_array [1] = 'green' // індекс 1 відповідає елементу green
 
 \$my\_array [2] = 'blue' // індекс 2 відповідає елементу blue 

Щоб отримати доступ до вмісту масиву, використовується ім'я масиву і індекс. Наступний код застосовується для виведення значень змінної \$my\_array.

\begin{verbatim}
 <!DOCTYPE html>
 <html lang = "en">
 <head>
   <title> Сторінка Web </title>
 </head>
 <body>
 <?php 

 \$my\_array = array ( 'red', 'green', 'blue');

 echo "Друге значення масиву -".  \$my\_array [1]. "<br/>";

 ?>

\end{verbatim}
  Перше значення масиву - red

\section{Асоціативні масиви}

Асоціативні масиви дозволяють використовувати більш корисні значення індексу. Для масивів з числовими індексами значення індексу створюються автоматично, починаючи з 0. Асоціативні масиви допускають застосування числових і строкових значень індексу. Символ між індексом і значеннями (=>) є знаком рівності, за яким відразу слід символ більше.

  \$Members = array( 'FName' => 'John', 'LName' => 'Smith', 'Age' => 50) 

У цьому прикладі члени масиву містять три елементи, проте використовуються рядкові індекси - FName, LName і Age.

 \$Members [ 'FName'] = 'John' // індекс FName відповідає елементу John
 
 \$Members [ 'LName'] = 'Smith' // індекс LName відповідає елементу Smith
 
 \$Members [ 'Age'] = '50' // індекс Age відповідає елементу 50 

Для доступу до вмісту масиву використовується ім'я масиву і індекс. Наступний код застосовується для виведення значень змінної \$members.

 

 The user's first name is John The user's last name is Smith
     The user's age is 50 

\section{Функції для роботи з масивами}

Крім функції array () система PHP включає безліч інших функцій для роботи з масивами. Наступний розділ описує деякі з найбільш часто використовуваних функцій. більш великий список доступний на Web-сайті PHP.

count() - функція count використовується для підрахунку числа елементів в масиві.

sort() - функція sort використовується для сортування елементів існуючого масиву.

shuffle() - функція shuffle використовується для випадкового перемішування елементів в заданому масиві.

sizeof() - функція sizeof є синонімом (аліасом) функції count ().

array\_slice (\$array\_name, offset, length) - функція array\_slice використовується для вилучення частини існуючого масиву. \$array\_name є ім'ям розрізається масиву, offset вказує позицію, де буде починатися розріз, length вказує число елементів, яке буде вирізано з масиву.

array\_merge (\$array\_name, \$array\_name) - функція array\_merge використовується для об'єднання або злиття двох або більшої кількості існуючих масивів. Імена масивів розділяються комами.



PHP включає також ряд зумовлених або глобальних масивів. Їх називають також суперглобального змінними, так як вони завжди присутні і доступні для всіх блоків сценарію PHP. Нижче показані зазвичай використовуються суперглобальні змінні PHP.

 \$\_GET[]
 \$\_POST[] 
 \$\_REQUEST[]
 \$\_COOKIE[] 
 \$\_FILES[] 
 \$\_SERVER[] 
 \$\_ENV[] 
 \$\_SESSION[] 

Суперглобальні змінні PHP будуть описані в подальшому. Масиви мають багато застосувань в PHP і програмуванні в цілому. Цей розділ представив деякі базові питання масивів PHP і описав деякі базові функції: це знадобиться при розгляді більш розвинених властивостей масивів в наступних розділах.
