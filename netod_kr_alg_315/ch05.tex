\item Виділення об’єктів та зв’язків між ними з використанням реляційної моделі даних для предметної області <<Інтернет-магазин взуття>>
\item Виділення об’єктів та зв’язків між ними з використанням реляційної моделі даних для предметної області <<Облік замовлень у сервісному центрі побутової техніки>>
\item Виділення об’єктів та зв’язків між ними з використанням реляційної моделі даних для предметної області <<Новини  приватного медичного кабінету <<Здравушка>>>>
\item Виділення об’єктів та зв’язків між ними з використанням реляційної моделі даних для предметної області <<Інтернет-магазин дитячих іграшок>>

\end{itemize}
